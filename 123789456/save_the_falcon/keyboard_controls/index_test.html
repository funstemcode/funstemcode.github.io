SAVE THE FALCON DIRECTIONS

  1. Right Click this link and select 'open in new tab' to get the javascript code Falcon Controls JS file
  2. Copy all of the code on the new page with 'ctrl + a' then 'ctrl + c'
  3. return to this index page and press 'ctrl + shift + j'
  4. Click 'sources' then 'snippets'
  5. Click 'new snippet'
  6. Click on the snippets window then press 'ctrl + v' to copy all of the javascript code
  7. press 'ctrl + s' to save the snippet
  8. press 'ctrl + r' to refresh the browser
  9. press 'ctrl + enter' to run the javascript code
